# 郑州出发：广州、仙本那、吉隆坡、重庆蜜月行程规划

版本：2026-07-01。核心目标是在婚礼后不过度折腾，先用广州过渡，再把重点放在仙本那海岛，最后用吉隆坡和重庆收尾。

## 推荐总方案

- 总天数：15 天 14 晚，2026-07-31 至 2026-08-14。
- 路线：郑州 -> 广州 3晚 -> 仙本那 7晚 -> 吉隆坡 2晚 -> 重庆 2晚 -> 郑州。
- 风格：广州负责城市休整和好吃好逛，仙本那负责蜜月核心体验，吉隆坡负责购物/城市夜景，重庆负责返程前的烟火气。
- 关键建议：优先锁定 08-03 广州直飞斗湖 AK1617；仙本那至少留 1 天机动，避免天气影响跳岛。

## 仙本那浮潜与潜水准备

### 你们没有浮潜经验时，先准备这些

- 面镜：提前在国内买能贴合脸型的面镜，比公用面镜更舒服。到货后在家用清水测试是否漏气。
- 呼吸管：选干式或半干式，第一次更友好。
- 水母衣/长袖防晒衣：仙本那日晒强，物理防晒比反复涂防晒更稳。
- 防晒：选择海洋友好型，SPF50+，但浮潜前仍要穿长袖。
- 溯溪鞋/潜水袜：登船、浅滩、珊瑚碎片区域更安全。
- 干袋和手机防水袋：证件、现金、房卡分开放。
- 晕船药：提前试药，不要第一次在船上才吃。
- 速干浴巾、替换衣物、保温薄外套：出水后风大会冷。

### 第一次浮潜当天的原则

- 穿救生衣，不为了拍照脱掉。
- 先在浅水区练呼吸，不急着追鱼群。
- 不踩珊瑚、不触摸海龟和海星。
- 两人保持视线距离，约定手势：OK、上船、需要帮助。
- 船员说水流强就不下水，留体力比硬玩重要。

### 若想学习深潜

- 体验潜水 DSD：半天到一天，不发证，适合试水。
- OW 开放水域证：通常 3 天，含理论、平静水域、开放水域训练。考证后 18 米内休闲潜水。
- 时间限制：潜水后通常 18-24 小时内不建议飞行，所以不要把深潜安排在 08-09 或 08-10 飞行日前。
- 身体条件：感冒、鼻塞、耳压平衡困难、熬夜、饮酒后都不适合潜水。
- 建议：这次以浮潜和体验潜为主；如果真想 OW，单独留 3 天纯学习，不和跳岛拍照日混排。

## 下单顺序

1. 先锁 08-03 广州 -> 斗湖直飞 AK1617。
2. 再锁 08-10 斗湖 -> 吉隆坡，确认仙本那最后一天不用早起赶船。
3. 复核 08-12 吉隆坡 -> 重庆，若中转太辛苦，考虑 08-13 或改飞广州/成都。
4. 定仙本那住宿：镇上 + 度假村/水屋组合优先。
5. 定广州、吉隆坡、重庆酒店。
6. 出发前 2 周定跳岛、接送、境外保险和电话卡。
